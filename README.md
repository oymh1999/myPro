# VoltMap 城市电车充电网络

VoltMap 是一个科技风的电车充电桩信息展示网站。页面以城市地图为核心，结合充电站点位、站点状态、可用枪数、功率、价格和排队时间等信息，模拟一个面向车主或运营方的充电网络监控台。

## 技术栈

- HTML5：页面结构与语义化布局
- CSS3：响应式布局、科技风视觉、地图底图、点位动效与面板样式
- 原生 JavaScript：站点数据管理、筛选搜索、地图点位渲染、详情联动和路线展示
- SVG：本地绘制城市路网、区域、水系和推荐补能路线

项目不依赖外部框架或 CDN，直接打开 `index.html` 即可运行。

## 功能情况

- 充电地图展示：通过自绘城市路网地图展示多个充电站点。
- 站点分类：支持超充、快充、慢充三类充电站视觉区分。
- 实时概览：展示当前筛选结果下的站点数、可用枪数和平均电价。
- 条件筛选：支持按充电类型、最少可用枪数、营业状态筛选站点。
- 关键词搜索：可按站点名称、区域或地址进行搜索。
- 站点详情：展示站点地址、营业状态、可用枪位、峰值功率、价格和预计等待时间。
- 附近站点列表：按可用枪数和距离排序，点击列表可联动地图选中站点。
- 推荐补能路线：基于可用站点模拟生成推荐路线，并支持开关显示。
- 响应式界面：适配桌面端和平板、手机宽度。

## 本地运行

方式一：直接打开

```text
index.html
```

方式二：使用本地静态服务器

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## 数据说明

当前站点数据为前端内置模拟数据，位于 `app.js` 的 `stations` 数组中。每个站点包含名称、区域、地址、充电类型、功率、总枪数、可用枪数、电价、等待时间、营业状态和地图坐标等字段。

后续可将该数组替换为真实接口数据，例如：

- 充电桩运营平台 API
- 城市公共充电设施开放数据
- 高德、百度、腾讯地图等位置服务数据
- 自建后台管理系统

## 可扩展方向
